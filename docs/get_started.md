# Get Started with Network Watchdog

Network Watchdog is a cross‑platform network monitoring tool that tracks live connections and detects suspicious activity.

## Requirements

* Python 3.8+
* pip
* Windows / Linux / macOS Terminal

## Quick Start

### Windows

```bat
NetworkWatchdogInstaller.bat
```

Choose a mode:

* Normal Mode
* Background Mode
* Extra Tools
* Logs

### Linux/macOS

```bash
chmod +x NetworkWatchdogInstaller.sh
./NetworkWatchdogInstaller.sh
```

You're ready to monitor network activity.
